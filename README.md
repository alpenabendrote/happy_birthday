# 特大ホームラン！お誕生日サプライズゲーム (Birthday Home Run Surprise)

ブラウザを開くだけで遊べる、お誕生日お祝い用の1画面ミニゲームです。
バットを振ってホームランを打つと、花火ともにスコアボードが設定した年齢に合わせて変化し、巨大なお祝いメッセージが飛び出します。
※コード内の歓声とか空振りとかは鳴らないかもしれません。

## 特徴 (Features)

- スタンドアロン: 画像や音声などの外部ファイルは不要。HTMLファイル1つだけで動作。

- プログラム合成音 (Web Audio API): 花火の爆発音（ドーン！）はすべてプログラム上で生成しています。

- パーティクル花火 (Canvas): HTML5 Canvas を使用し、花火を描画します。

- 満員のスタジアム演出: 遠景のシルエットと数千の揺らぐ点群を描画し、スタジアムの熱気を表現しています。

- スコア演出: ホームランの確定後、スコアボードが「x」から「x+1」にカウントアップする仕掛けが入っています（デフォルトは8歳のお誕生日を想定）。

## 遊び方 (How to Play)

happy_birthday.html をダウンロードします。

PCやタブレットのブラウザ（Google Chrome, Edge, Safariなど）でダブルクリックして開きます。

ボールが飛んでくるので、タイミングよく画面内のバットをクリック（またはタップ）してスイングします。

見事ホームランを打つと、お祝いのショーが始まります！

## カスタマイズ方法 (Customization)

ご自身のお子さんやご友人向けにアレンジする場合は、コード内の以下の部分をテキストエディタで書き換えてください。

### 1. お祝いメッセージと名前の変更

HTML内（<div id="scoreboard">の直後）にある以下の部分を書き換えてください。

```
<div id="main-message">
  特大ホームラン！！<br>
  <div id="sub-message">〇〇、お誕生日おめでとう！</div>
</div>
```

### 2. 年齢（スコア）の変更

お誕生日の年齢に合わせて、スコアが変わる演出を変更できます。

① 初期スコアの変更（HTML下部）:
現在の年齢（例：7歳なら 7）に設定します。

```
<div class="score-team"><span>B-DAY</span><span id="bday-score" class="score-val">7</span></div>
```

② ホームラン後のスコアの変更（JavaScript内）:
新しい年齢（例：8歳なら 8）に設定します。コードの下の方にある startFireworksShow 関数の中を探してください。

```
// ★1.5秒後にスコアを変更！★
setTimeout(() => {
  bdayScore.style.color = '#FF1493';
  bdayScore.style.textShadow = '0 0 20px #FFF';
  bdayScore.innerText = '8'; // ★ここを新しい年齢に変更★
  // ...
```
